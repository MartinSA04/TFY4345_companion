# TFY4345 — Klassisk mekanikk

Interaktiv studieguide for [TFY4345](https://www.ntnu.no/studier/emner/TFY4345)
ved NTNU: **[klasmek.martinsundal.no](https://klasmek.martinsundal.no)** — moduler med quiz, flashcards,
formelsamling og begreper.

Bygget på [StudyCompanion](https://github.com/MartinSA04/StudyCompanion);
dette repoet inneholder kun innhold (`content/`).

## Kjør lokalt

```bash
pnpm install
pnpm dev
```
